# Khitai
### Recon and Scan toolset for pentesting engagements

##### Under construction!

> Installation:  
`git clone https://github.com/rafaelbaldasso/Khitai`  
`cd Khitai`  
`sudo ./install.sh`  
  
> Usage:  
`sudo ./Khitai.sh`  
  
> Results are printed to the terminal and saved to a file at the current directory.
